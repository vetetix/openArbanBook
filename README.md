# openArbanBook

## About
A recreation of Arban's Cornet Method
This is the primary goal of the project at this time. Until there are more collaborators then it is not expected to be completed any time soon...

## Downloads
You can download a copy of the Open Arban Book in PDF format, as it is right now, from the [Neoteric Brass Website](https://neotericbrass.com/downloads/openarbanbook/).
Be aware that the book is incomplete and may also contain errors, feel free to let me know what you find or make a PR!

## News
The project was started in 2020. At the time the oldest version of Arban's Grand Method that was easy to find for transcription was the 1879 Pepper/Rollinson edition. It was chosen for use in creating the Open Arban Book with the hope that it would be the most authentic version. Since the prject was resurrected in 2026 it has come to light that the actual first edition of the Grande Methode has been scanned by the Bibilotheque nationale de France and is now available on IMSLP. It may seem on the surface that this renders the Open Arban Book a little redundant, but the important value of this project is to create a book that can be freely edited and redistributed without copyright concerns, so the work continues...

From this point on oAP will be using the first edition (1864) of the Arban book for transcription. In future time will need to be spent going back through completed pages to check for markings that are not accurate, but at this stage it isn't considered hugely important.
